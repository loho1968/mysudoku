## 今日完成

### 一、功能开发（commit `32684b2`）

#### 新增能力

1. **本地环境判定**（`src/lib/env.ts`）
   - `isLocalEnv()`：客户端按 `window.location.hostname` 判定（localhost/127.0.0.1/::1）
   - `isLocalRequest()`：服务端按请求 Host 头判定
   - `LOCAL_DEV_HEADER = "X-Local-Dev"`：本地旁路标识头

2. **本地免密维护模式**
   - `verifyEditMode`：本地请求 + `X-Local-Dev: 1` 双重确认 → 放行（无需密码）
   - `useEditMode`：本地环境 `toggleEditMode` 直接进入编辑模式，不弹密码框
   - `apiFetch`：本地环境自动注入 `X-Local-Dev: 1` 头
   - `/puzzles` 页面：本地环境首次进入时自动切换为维护模式

3. **AppHeader 按钮调整**
   - `/puzzles` 路径下隐藏「选择题目 / 数独技巧 / 题库维护」三个出题入口按钮
   - 其他页面（如 `/game`）保持原样

4. **题库同步 UI**（`src/components/PuzzleManager/SyncBar.tsx`）
   - 仅本地环境渲染，部署到服务器返回 null
   - UI：服务器地址 Input + 密码 Input.Password + 拉取按钮 + 同步按钮
   - 地址/密码记忆于 localStorage（`mysudoku_sync_server` / `mysudoku_sync_password`）
   - 完整的错误处理：401 密码错 / 403 权限 / 网络错误分别提示

5. **新增 3 个 API**
   - `GET  /api/puzzles/all`：全量导出（puzzles + tags，预读避免 N+1）
   - `POST /api/puzzles/upsert`：批量 upsert（事务，按 ID 主键，含关联表重建）
   - `POST /api/tags/upsert`：批量 upsert 标签

#### 同步流程（按 ID upsert）

- **拉取（服务器 → 本机）**：
  服务器 `GET /api/puzzles/all`（带 X-Edit-Password）
  → 本地 `POST /api/puzzles/upsert`（apiFetch 自动加 X-Local-Dev）

- **同步（本机 → 服务器）**：
  本地 `GET /api/puzzles/all`
  → 服务器 `POST /api/puzzles/upsert`（带 X-Edit-Password）

### 二、部署上线（腾讯云 PM2）

#### 部署踩坑

1. **服务器 git 状态异常**：`/opt/mysudoku` HEAD 停在 `a807cbb`（旧提交），但工作区
   文件已是后续 commit 内容（疑似历史 scp 部署）。处理：用户确认后
   `git reset --hard origin/master`，数据库备份到 `data/mysudoku.db.bak-20260720-114653`
   后保留完好。

2. **孤儿文件导致 build 失败**：服务器上有未跟踪的 `src/components/ChainBuilder/`、
   `src/components/SudokuGrid/ChainCanvas.tsx`、`src/hooks/useChainDrawing.ts`
   （历史 scp 残留），TypeScript 编译时扫到这些文件引用不存在的 `ChainType` 类型，
   导致 build 失败。处理：`git clean -fd` 删除（这些文件不在 git 仓库，本地也没有）。

3. **服务器实际密码 ≠ 代码默认值**：服务器 `settings.edit_password = 'mysudoku'`
   （用户早期库的值），而 `db.ts` 中新默认是 `Kili+2016`。`INSERT OR IGNORE`
   不会覆盖已有值，所以服务器保留旧密码。**用户在 SyncBar 输入密码时要用 `mysudoku`**。

4. **pm2 端口**：实际监听 3003（不是 PLAN.md 文档里的 3000）。
5. **nginx**：mysudoku 没配反代，目前通过公网 IP:3003 直连（`http://110.42.225.37:3003`）。
