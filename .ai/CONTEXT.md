## 项目概况

**mysudoku** 是一个个人学习和练习数独的单页面网站。

**核心定位:** 服务于个人数独技巧练习，而非大众化在线数独游戏。重点在于辅助学习解题技巧（通过标签分类、智能笔记、强弱链图等功能）。

## 目标用户

- 自己（个人使用）
- 可能分享给同好

## 非目标

- 不追求多用户/社交功能
- 不求商业化
- 不求题目自动生成（阶段一）
- 不求题目自动难度评定

## 技术约束

- **运行环境:** 腾讯云轻量服务器 OpenCloudOS 9.2，Node.js v24
- **数据库:** SQLite（单文件数据库，无额外服务依赖）
- **部署:** PM2 进程守护，可能后续加 Nginx 反代
- **开发:** 两台 Mac（公司+家里），通过 SSH 连接服务器远程开发
- **代码仓库:** GitHub (loho1968/mysudoku.git)

## 设备场景

| 场景 | 说明 |
|------|------|
| 日常做题 | 桌面端浏览器（大屏幕，可调节字体） |
| 碎片时间 | 手机端浏览器（PWA 安装到桌面，离线可用） |
| 题目管理 | 桌面端编辑模式（粘贴导入、标签维护） |
| 服务器部署 | PM2 常驻，SSH 远程开发 |

## 参考项目/灵感

- Sudokuexchange.com — 高亮系统设计参考
- HoDoKu — 解题技巧分类参考
- SudokuWiki — 强弱链可视化参考
